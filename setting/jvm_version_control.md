# 버전 관리

## Java 버전 전환 설정

- `~/.zshrc`에 Java 버전 변경 함수 추가

```shell
function javahome() {
  export JAVA_HOME=$(/usr/libexec/java_home -v "${1:-1.8}");
  java -version
}
```

- java version 확인

