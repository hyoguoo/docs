# Intellij Setting

## 라이브 템플릿 생성

`Setting` - `Edit` - `Live Templates` - `+`

- `Group`: 템플릿 그룹(실제 동작 범위와는 무관)
- `abbreviation` : 단축어
- `Template Text`: 템플릿 내용(변수는 `$VARIABLE$`로 표현)
- `Description`: 템플릿 설명
- `applicable`: 적용할 언어 및 범위
- `Edit Variables`: 템플릿 내용에서 사용할 변수 설정
