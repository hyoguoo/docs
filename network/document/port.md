# PORT

> 한 번에 둘 이상 연결하기 위한 기능으로, 같은 IP 주소 내에서 프로세스 구분

- ip: 목적지 서버를 찾는 것
- port: 목적지 서버 내에서 프로세스를 찾는 것

## PORT 번호

- 0 ~ 65535 할당 가능
- 0 ~ 1023 : Well-known port
    - FTP(21), Telnet(23), HTTP(80), HTTPS(443), SMTP(25), POP3(110), IMAP(143)
